# ML Models

A collection of machine learning algorithms implemented from scratch for educational purposes. Each model includes comprehensive tests using scikit-learn datasets.

## Implemented Models

-Decision Tree
-Random Forest
-Gradient Boosting
-XGBoost
-Linear Regression

## Project Structure

```
ml_models/
├── model/
│   ├── __init__.py
│   ├── decision_tree.py
│   ├── random_forest.py
│   ├── gradient_boosting.py
│   ├── xgboosting.py
│   └── linear_regression.py
├── test/
│   ├── test_decision_tree.py
│   ├── test_random_forest.py
│   ├── test_boosting.py
│   └── test_linear_regression.py
└── README.md
```

## Requirements

- numpy
- scikit-learn (for datasets and evaluation metrics only)
